Help make Beams better

Beams is an open full-stack framework for building secure websites. It's free to use under MIT License.

Roscoe Labs is the lead developer of Beams, though we heartily encourage and welcome contributions of all kinds to make the software better.

There are several ways for technical and non-technical contributors alike to participate. The best ones are listed on this page, which we will update from time to time.

Community Resources

  • Download. A free copy of the source code for Beams v0.2.5, the latest release, is available for download here via Roscoe Labs.
  • GitHub. Our workhorse tool for all things development-related. It can be used to view the main code repo for Beams, report bugs or other issues, view release notes and history, or contribute to the project wiki. In particular, the wiki includes a getting started guide that newcomers to Beams may find useful.
  • Listserv. The Beams Working Group is an email-based forum for coders and non-coders alike to discuss the framework, ideas for improving it, et cetera.
  • Donations. Coming soon. We're going to add links to donate money online to support open development of Beams.
